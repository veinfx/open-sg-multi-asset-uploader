***
## How to use

1. 프로젝트를 콤보박스에서 선택한다.
2. Asset  types 콤보박스 에서 type 을 선택한다.
3. Asset dir open 버튼클릭하여 샷그리드에 업로드할 에셋파일들이 있는 디렉토리를 선택한다.
4. Asset Upload 버튼을 클릭하면 선택한 프로젝트와 asset type 으로 디렉토리안의 모든 파일들이
   샷그리드에 에셋이 업로드된다. 
5. close 버튼은 ui 창이 닫힌다.
